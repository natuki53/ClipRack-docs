機能

Windows準拠の操作感をベースに、macOSの“馴染み”も崩さない設計。

グローバルホットキーで表示/非表示をトグル。
↑↓

↑↓ / PageUp / PageDown で高速に選択。
⏎

Enter または単クリックで即ペースト（既定）。

使い方

01
呼び出す

ショートカットキー ⌘⇧V。

02
選ぶ

↑↓（PageUp/PageDownもOK）で履歴を選択。

03
貼り付ける

Enter または 単クリックで貼り付け。

04
固定する

右端の ピンマーク でピン留め。

インストール（一般ユーザー向け）
  1. ダウンロード（Releases から）
  2. アプリを アプリケーション フォルダへ移動
  3. 起動して、案内に沿って必要な権限を付与

アクセシビリティ権限について

選んだ履歴を 自動で貼り付け するために、アクセシビリティの許可が必要です。

許可がある場合
  • ホットキー（⌘⇧V）で呼び出し
  • ↑↓で選択、Enter/クリックで貼り付け
許可がない場合
  • 画面内メニューで手順を案内します
  • 手順に従って許可を行ってください
許可手順
  1. システム設定 → プライバシーとセキュリティ → アクセシビリティ
  2. ClipRack を許可
